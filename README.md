#  COVID-19 Data Analysis Dashboard

This project analyzes global **COVID-19** data using **SQL**, **Python**, and **Streamlit** to uncover trends in cases, deaths, vaccinations, and testing. The goal was to transform raw data into meaningful insights through data cleaning, analysis, and an interactive dashboard.

##  What I Did

* Cleaned and prepared COVID-19 datasets for analysis.
* Used SQL to answer analytical questions and identify trends.
* Analyzed cases, deaths, recoveries, testing, and vaccination data across countries.
* Created visualizations to compare the impact of COVID-19 over time.
* Built an interactive Streamlit dashboard for easy data exploration.

##  Tech Stack

* SQL
* Python
* Pandas
* Matplotlib
* Streamlit
* Git & GitHub

##  Key Takeaway

This project showcases an end-to-end data analysis workflow, from cleaning and querying large datasets to building an interactive dashboard that helps visualize the global impact of COVID-19.
